# `skills`

Reference home for Savant skill templates, local fixtures, registry examples, and authoring guidance.

## Important boundary

Tenant-authored production skill content is expected to live in a **connected external Git environment**, not primarily inside this repo.

That means this directory is used for:

- reference templates
- local development fixtures
- validation examples
- bootstrap layouts for newly provisioned tenant repositories

See `skills/templates/tenant-skill-repo/` for the recommended starter layout for external tenant repositories.
